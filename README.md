# MIDI-LED
Using a Raspberry Pi to control LED lights with MIDI output

# Running on your RPI
sudo pip3 install adafruit-circuitpython-neopixel
git clone https://github.com/jzarzycki/MIDI-LED.git
cd MIDI-LED
sudo python3 midi-led.py
